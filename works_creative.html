    SCRAPER/AI Instagram Scraper
    SCRAPER

    Instagram Scraper

    A scraper that scrapes followers of an Instagram user, tracks their comments, and generates comments on a post by utilizing AI.

    • Scrapes followers of an Instagram user
    • Can go through posts of the followers to create their profiles using textual information and image of the post (Deepface,Object Detection Model and LLM used )
    • Can track the posts of a user
    • Can generate comments for the user leveraging AI
    • Proxy is used to bypass blocking
    • Admin Panel in svelte to start and monitor the scraper

    This tool can be utilized for follower analysis, enabling segmentation and categorization of an Instagram user's followers based on their profiles and activities. Additionally, it has the capability to generate comments using AI based on the picture posted on instagram, facilitating efficient and targeted engagement with the user's audience.

    My Role:

    Developed the system using a range of isolated microservices, including a FastAPI-based backend that orchestrates operations by calling APIs. A scraper microservice was created to generate a user's Instagram follower list and scrape each follower's profile to build a database. A pub-sub system using RabbitMQ is implemented to distribute workloads across multiple workers, each operating as an agent in a separate VM. Additionally, another microservice was developed to generate comments. This service runs a cron job to scrape specified users' data, detect new posts, and analyze the text and images of these posts. Image analysis is performed using DeepFace to gather information about the person in the post (such as gender, age, and race) and RAM (Recognize Anything Model) to detect objects in the image. Comments are generated based on this analysis using LLM (LLaMA) and then posted using another microservice built by the organization. RabbitMQ is also utilized to distribute the comment generation workload.

    Stack Used:

    FastAPI,Microservice Architecture, Selenium(includes proxy bypass using smartproxy), Svelte, Tailwind CSS, LLM (Llama), Deepface, Recognize Anything Model (RAM), Mongodb, RabbitMQ
